# **App Name**: XFuse Portal

## Core Features:

- AI Role Assistant: Provides AI-powered assistance tailored to each role, offering solutions, content drafts, and task-specific support using a Gemini API tool.
- Interactive Roleplay Dojo: Simulates real-world client interactions for training purposes. The tool generates scenarios and evaluates user responses, providing instant feedback to improve communication skills.
- Performance Evaluation AI Report: Generates AI-driven performance evaluation reports, including summaries of strengths and weaknesses, and proposes SMART goals for improvement based on evaluation metrics.
- Client Health Analysis: Uses AI to analyze client health based on feedback and other metrics, providing insights and action plans to improve client relationships and satisfaction using a Gemini API tool.
- AI Template Generator: Generates custom marketing templates based on user-defined contexts and types, such as client replies, briefs, content, and ad copies.
- SOP (Standard Operation Procedures) Card: Displays role-specific responsibilities and daily/weekly tasks in a clear and concise format, along with a list of KPIs to make sure the employee knows how they will be measured.
- Centralized Navigation and Analytics: Provides a dashboard with key performance indicators (KPIs), leads tracking, client health monitoring, and a template library for streamlined marketing operations.

## Style Guidelines:

- Primary color: Indigo (#6366F1), evoking trust and professionalism.
- Background color: Light gray (#F9FAFB), providing a clean and neutral backdrop.
- Accent color: Purple (#A855F7), complementing indigo and adding a touch of creativity.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined look.
- Code font: 'Source Code Pro' (monospace) for displaying code snippets.
- Consistent use of Lucide React icons throughout the app, providing a clean and modern visual language.
- Subtle animations, such as fade-ins and loading spinners, to enhance user experience without being distracting.
